# NPPK Website

A modern political campaign website built with Laravel and the Miata template.

## Features

- Responsive design
- Modern UI/UX
- Interactive components
- Campaign management
- Event scheduling
- Team showcase
- News and updates section

## Technical Stack

- Laravel 10.x
- Bootstrap
- jQuery
- Font Awesome
- Owl Carousel
- WOW.js
- Nivo Slider
- Mean Menu

## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/nppk-website.git
```

2. Install dependencies
```bash
composer install
npm install
```

3. Configure environment
```bash
cp .env.example .env
php artisan key:generate
```

4. Run migrations
```bash
php artisan migrate
```

5. Start the development server
```bash
php artisan serve
```
